fun VGSCollect.cardExpirationDate(view: ExpirationDateEditText)

Binds the card expiration date input view and configures it for Card Management requests.

The expiration date will be serialized into separate exp_month and exp_year fields. The visible format will be MM/yy.

Parameters

view

The ExpirationDateEditText instance used to input the expiration date.
